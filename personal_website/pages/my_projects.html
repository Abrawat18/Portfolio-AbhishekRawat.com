College Projects

School/College Bus route Determiner(Travelling Salesman Problem)


I undertook final year’s project ‘School/College Bus Route Determiner (Based on Travelling Salesman Problem)’ which was developed with the purpose of providing various facilities at a single and coherent platform. It provided users the facility of seeing optimal route between different stations. This utility can be used in many organisations ranging from education institutes to police vehicles.
Environment: JavaScript, Google API, HTML, JSP, Servlet.

Vehicle Management system


I developed a project on ‘Vehicle Management system’ using Visual Basic 6.0, Oracle 8i database with ADODB and ADODC connectivity. We provided some fascinating features like enabling any car user to search and communicate with any car service centre in domain, getting an appointment for servicing, test drive and accessories to purchase from dealer.
Environment: JavaScript, HTML, Visual Basic, PL/SQL.
